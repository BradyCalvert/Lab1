Please place all raw textures in Textures folder.
Create materials in Materials folder.
Prefabs should go in the prefabs folder. Create a new folder inside of prefabs for each scene the prefab will reside in .
Scripts should go into the scripts folder. Create a new script for scrips that are a common function.
Scenes should be saved into the scenes folder. Create a new folder inside scenes for scenes that share a common function, such as Menu Scenes
Animation clips belong inside the Animations folder. create a new folder for each time of animation. Controllers are created from those clips and should go in the animation Controllers folder.
